# ZDK Homebrew Tap

Homebrew tap repo for my software

## Installation

```bash
brew tap zdk/tools
```

## Software

### lowfat

Low-fat command output for LLM environments. Strips noise, saves tokens.

```bash
brew install lowfat
```

### rm-safely

```bash
brew install rm-safely
```

### distrowat

```bash
brew install distrowat
```
